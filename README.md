# Role and Opportunity Model

## What this repo does

This repository contains a deterministic TypeScript MVP for evaluating WR and TE receiving roles. It scores role quality, team opportunity environment, role stability, and vacated opportunity. It does **not** predict fantasy points.

## Current MVP scope

- WR/TE-only role intelligence.
- Typed interfaces for player roles, team context, and evaluation output.
- Deterministic archetype classification and score generation.
- Seeded scenarios for quick validation and iteration.

## How to run

```bash
npm install
npm run dev
```

## How to test

```bash
npm test
```
